ðŸ“‹ Step-by-Step Setup Instructions

1Download & Install OBS Studio

If you don't have OBS Studio installed:

  • Visit: https://obsproject.com/
  • Download the version for your operating system
  • Install and launch OBS Studio

2Open Stream Settings

In OBS Studio:

  • Click Settings in the bottom-right corner
  • Select Stream from the left sidebar

3Configure Stream Settings

Set the following values:

  • Service: Select "Custom..." from the dropdown
  • Server: Enter rtmp://localhost:1935/live
  • Stream Key: Enter test-stream-key
Note: You can generate a new stream key in the test interface if needed.

4Configure Output Settings (Recommended)

For optimal streaming quality:

  • Go to Settings â†’ Output
  • Set Video Bitrate: 2500 Kbps
  • Set Audio Bitrate: 160 Kbps
  • Set Encoder: x264
  • Set Keyframe Interval: 2 seconds

5Add Video Sources

Add content to stream:

  • In the Sources panel, click the + button
  • Choose a source type:
    • Display Capture - Capture your entire screen
    • Window Capture - Capture a specific window
    • Video Capture Device - Use your webcam
  • Configure the source settings as needed

6Start Streaming

Ready to go live:

  • Click Start Streaming in the bottom-right corner
  • OBS should show "Streaming" status
  • Check the test interface for stream detection
Success! Your stream should now appear in the monitoring dashboard.
